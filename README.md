# OneQueueHistory_Server
한끝 한국사 서버_RESTFUL API

## 주제
유아용 간단한 게임 제작

## 핵심 기능
> 마르코브 체인을 활용한 문장 제작
> 단어 맞추기
> 문장 맞추기

## 간략한 구조도

## UI
* 로그인 및 회원가입
* 회원 관련 기능
* 회차별 문제풀기
* 랜덤 문제풀기
* 빈 칸 맞추기
* 오답노트
* 취약점 분석
